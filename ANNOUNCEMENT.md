# MarkHome Launch Copy

## LinkedIn

I just published MarkHome v0.1.

MarkHome is Markdown for simple 2D home layout diagrams.

You write this:

```txt
home "My Apartment" unit cm
room LivingRoom at 0,0 size 420x360
window LivingRoom north at 110 size 180
door LivingRoom east at 155 size 90
item sofa in LivingRoom at 55,245 size 230x75
```

And MarkHome renders it as a clean SVG floor plan.

The goal is not to build a CAD tool, 3D planner, BIM replacement, or interior design marketplace. The goal is a small, readable, Git-friendly text standard for home layouts.

v0.1 includes:

- parser
- SVG renderer
- npm package
- CLI
- playground
- docs
- initial language spec
