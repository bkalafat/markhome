# Changelog

## 0.1.0

Initial public release.

- Added MarkHome v0.1 parser and AST types.
- Added SVG renderer.
- Added browser auto-render API with `initialize({ startOnLoad: true })`.
- Added `render(source)`, `renderSvg(ast)`, and `parse(source)`.
- Added CLI renderer for `.markhome` to `.svg`.
- Added React playground.
- Added documentation site.
- Added examples and initial specification.
